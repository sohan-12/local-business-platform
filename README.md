# Local Business Platform

A full-stack web application to help users discover, review, and manage local businesses.

## Tech Stack
- Frontend: React, HTML, CSS
- Backend: Node.js, Express
- Database: PostgreSQL
- Authentication: JWT, bcrypt

## Features
- User registration and login
- Role-based access (Admin, Owner, User)
- Business listing and approval
- Reviews and ratings
- Admin dashboard
